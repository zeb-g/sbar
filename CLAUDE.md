## Project overview
This is a static GitHub Pages site at soundbars.zebronics.com (also live at zeb-g.github.io/sbar/).
The page is a recommendation engine for 37 Zebronics Juke Bar soundbars.

## File map
| File | Purpose |
|---|---|
| `index.html` | Full landing page — all UI, JS, CSS in one file |
| `products.json` | 37 soundbar SKUs — prices, ASINs, FSNs, specs, buy URLs |
| `scores.json` | Recommendation scores — generated by convert_scores.py |
| `Soundbar_Scores.xlsx` | Source scorecard — edit this, then run convert_scores.py |
| `convert_scores.py` | Converts Excel → scores.json |
| `CNAME` | soundbars.zebronics.com |

## Architecture
Static GitHub Pages. No backend, no build step, no npm.
- `index.html` fetches `./products.json` and `./scores.json` at runtime
- Edit data in Excel → run convert_scores.py → commit JSON → push → live in 60s

## How to update scores
1. Edit `Soundbar_Scores.xlsx` (scores 1–5, never outside this range)
2. Run: `python convert_scores.py`
3. Verify output: ✅ scores.json written — 37 products
4. Commit and push: `git add scores.json && git commit -m "Update scores" && git push`

## How to update a price
Edit `products.json` — find by product id, change `price`, `mrp`, or `save` field.
Then: `git add products.json && git commit -m "Update [product] price" && git push`

## Scoring scale
1 = worst fit, 5 = best fit
Dimensions: room (studio/small/medium/large), tv (tv_small/tv_medium/tv_large/tv_xlarge),
use (movies/gaming/sports/everyday/music)
